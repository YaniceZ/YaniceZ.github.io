Coastal Ecosystem Shifts: Cable Bay

Objective

Monitor land cover changes using Sentinel-2 imagery to support conservation efforts.

Methods

  • Processed annual cloud-free Sentinel-2 Level-2A composites
  • Applied Random Forest classification (100 trees, 255 polygons)
  • Conducted transition matrix and hotspot analysis

Outcome

  • Achieved 97.0%-98.6% accuracy
  • Identified critical ecosystem changes
  • Supported UN SDG 15 and BC's Biodiversity Strategy
